* *Familiarity with AWS core services* (IAM, EC2, S3)
* *Experience with command-line tools* (bash, git)

=== AWS services used

This workshop integrates with the following AWS services:

* *Amazon CloudWatch* - Centralized logging and monitoring
* *AWS IAM* - Identity and access management
* *AWS STS* - Security Token Service for temporary credentials
* *Amazon S3* - Object storage for Loki log storage
* *Amazon DynamoDB* - NoSQL database for application data
* *AWS Secrets Manager* - External secrets management

=== OpenShift components used

You will work with these OpenShift operators and components:

* *OpenShift Logging 6.x* - Log collection and forwarding
* *Loki Operator* - Efficient log storage backend
* *Cluster Observability Operator* - Logging UI integration
* *Secrets Store CSI Driver* - External secrets mounting
* *Source-to-Image (S2I)* - Container image building
